项目简介

MyRpcDemo

开源项目
一款基于 Nacos 实现的 RPC 框架。网络传输实现了基于 Java 原生 Socket 与 Netty 版本，并且实现了多种序列化与负载均衡算法，系统手写代码 2.2k 余行。
  • 技术栈：Netty + Nacos
  • 实现了基于 Java 原生 Socket 传输与 Netty 传输两种网络传输方式
  • 实现了两种序列化算法，Json 方式、Kryo 算法（默认采用 Kryo方式序列化）
  • 实现了两种负载均衡算法：随机算法与轮转算法
  • 使用 Nacos 作为注册中心，管理服务提供者信息
  • 消费端如采用 Netty 方式，会复用 Channel避免多次连接
  • 如消费端和提供者都采用 Netty 方式，会采用 Netty 的心跳机制，保证连接
  • 接口抽象良好，模块耦合度低，网络传输、序列化器、负载均衡算法可配置
  • 实现自定义的通信协议
  • 服务提供侧自动注册服务

BiliLottery

开源项目
B 站抽奖小程序，支持即刻抽奖、定时抽奖，基于云服务器和代理池的 B 站网络接口调用，抽奖对象包括视频、动态、文章，同时基于洗牌算法实现公平、高效抽奖。
  • 技术栈：微信小程序 + SpringBoot + MySQL
  • 分析视频、动态、文章、Reply 等实体的属性，设计了良好的继承结构
  • 网络接口调用基于云服务器和代理池，保证网络请求的可靠性，充分考虑 B 站接口的各类响应代码，保证系统的鲁棒性；
  • 通过对网络请求加锁、复用可用的 client 实例，来防止重复实例化 HttpClient，避免资源浪费
  • 能够根据评论总数采取自适应抽奖方式，包括全量、非全量抽奖，最大程度保证系统的低延迟和抽奖的公平性之间的平衡
  • 定时抽奖：基于 ScheduledThreadPoolExecutor，通过内存定时线程池 + 数据库存储定时任务的模式，维护系统唯一时间戳，提高系统处理定时任务的并发数
  • 系统手写代码5.5k余行，目前小程序用户340+人
